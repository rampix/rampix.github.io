About the Rampix Modpack

Rampix is a modpack created for older computers without a dedicated graphics card. Optifine makes it able to run the newer mods (see below) without any FPS lag.

  • Forge
  • Forestry
  • Thaumcraft3
  • Optifine
  • EE3
  • Modular Powersuit
  • NEI
  • IC2
  • Portal Gun
  • Zan's Minimap
  • Anti-Griefing
  • BuildCraft
  • TreeCapitator
  • Inventory Tweaks

Running on Minecraft Version 1.5.1
